Univa Nova is a paid commercial font (Fontastica / MyFonts) and is not
available on Google Fonts or any free/legal CDN. It was not bundled here.

If you have a licensed copy, drop the webfont files here with these exact
names (see src/index.css for the @font-face rules that reference them):

  UnivaNova-Regular.woff2 / .woff   (weight 400)
  UnivaNova-Medium.woff2  / .woff   (weight 500)
  UnivaNova-SemiBold.woff2/ .woff   (weight 600)
  UnivaNova-Bold.woff2    / .woff   (weight 700)

Until then, the app falls back to Inter / system sans-serif, which is
visually close (geometric sans).
